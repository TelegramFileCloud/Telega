Get Account Info

Add imports:

using Telega.Client;
using Telega.Rpc.Dto.Functions.Users;
using Telega.Rpc.Dto.Types;

Print your account info:

var myUser = new InputUser.SelfTag();
var myInfo = await tg.Call(new GetFullUser(myUser));
Console.WriteLine(myInfo);
